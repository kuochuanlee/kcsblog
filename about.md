# 關於本站

歡迎來到 **KC's Blog**。這裡是我記錄技術隨筆、生活點滴與靈感的地方。

## 建立初衷
這個部落格基於 **VitePress** 構建，追求極簡的視覺體驗與高效的閱讀感受。我喜歡黑灰白與奶茶色的沈穩質感，並致力於分享有價值的內容。

## 技術棧
- **框架**: VitePress (Vue 3 + Vite)
- **風格**: 純 CSS 打造的極簡主義
- **搜尋**: 支援 CJK (中日韓) 分詞的原生搜尋

## 未來規劃

- AI洞察分析
