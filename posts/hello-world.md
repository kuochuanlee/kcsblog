# 哈囉！這是我的第一篇 VitePress 文章

歡迎來到我的極簡部落格。這是使用 **VitePress** 建立的第一篇文章。

## 為什麼選擇 VitePress？

1. **極速**: 基於 Vite，開發與建置都非常快。
2. **Vue 3 驅動**: 可以在 Markdown 中直接使用 Vue 組件。
3. **優雅的主題**: 內建主題已經非常漂亮且支援深色模式。

## 測試圖片

如果你有圖片，可以放在 `public/images/`，並使用標準 Markdown 語法引用。

## 程式碼測試

```typescript
function hello(name: string) {
  console.log(`Hello, ${name}!`);
}
```

這是一段測試摘要長度的文字，這篇文章將會被 `posts.data.ts` 自動掃描，並計算出適合的摘要內容與預估閱讀時間。
